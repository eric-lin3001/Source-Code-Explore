# MY THREADS

## Process(进程) vs Threads(线程)

- 我的理解：进程和线程是包含关系，一个进程可以包括多个线程。

- 进程可以理解为一个应用程序，线程可以是这个程序里的子任务。
- 进程和线程都是可以同时进行的（可以多进程，也可以多线程）
  - 多进程举例：在一台主机上同时打开word和excel
  - 多线程举例：打开“网易云音乐”这个应用程序（进程），可以边听歌（线程1）边搜索音乐（线程2）。
